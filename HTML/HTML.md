



## HTML语法与基础标签

### 认识标签

HTML是网页使用的语言，网页的HTML代码就是有许许多多不同的标签（tag）构成。HTML是网页开发的基础，定义了网页的**结构和内容，**浏览器访问网站，起始就是从服务器下载了HTML代码，标签是用来告诉浏览器，如何处理这段代码。标签的内容就是浏览器索要渲染的，展示在网页上的内容。

HTML全名叫做 "超文本标记语言"，特点：**支持超链接**，点击链接就可以跳转到其他网页，从而构成整个互联网。超文本标记就是标签。

#####     成对标签       

**title 标签**

​    标签放在一对尖括号里面（比如开始标签`<title>`)，结束标签在开始标签名之前加斜杠（比如`</title>`（标签对大小写不敏感，只是通常小写）,开始标签和结束标签称为标签对。

```html
 <title>网页标题</title>
```

##### 单标签

单标签，标签本身就足够完成功能，不需要标签之间的类容。实际应用中，他们主要用来提示浏览器，做一些特别的处理。

```html
<mete charsrt="UTF-8"
```



### 基础标签

#### 标题h标签（body里面使用的）

h系列标签表示“标题“语义，h是headline的意思

| 标签 | 语义     |
| :--: | :------- |
|  h1  | 一级标题 |
|  h2  | 二级标题 |
|  h3  | 三级标题 |
|  h4  | 四级标题 |
|  h5  | 五级标题 |
|  h6  | 六级标题 |

注意：logo等重要内容要放在`<h1></h1>`中。`<h1></h1>`标签一般一个网页只能放置一个，否则会被引擎视为作弊。



#### 段落p标签（paragraph)

p标签里只能放**文字**、图片、**表单**等元素，不能放置h系列标签，也不能放置其他的p标签。任何段落都要放在p标签中。

#### div标签

div标签division表标示**分割**，相同内容放到一个div标签对里面，以和其他内容分割开。

> 1. 网页头部，轮番图，文章内容都要放在一个div标签里面。
>
> 2. div是最常见的html标签，因为他可以结合css使用，实现网页的布局，这种布局形式叫做DIV+CSS
> 3. div像是一个容器，什么都可以容纳，因此工程师也习惯称呼div为盒子。
>
> 



div的常见分类：div标签可以添加class属性表示“类名”，类名服务于css（后面章节）

|   区域   | 类名   |
| :------: | ------ |
|   页头   | header |
|   logo   | logo   |
|  导航条  | nav    |
|   横幅   | banner |
| 内容区域 | conter |
|   页脚   | footer |

标签也可以嵌套,但必须保证正确的闭合顺序，不能跨层嵌套

```html
<div>
    <p>
        hello word
    </p>
</div>
```

### 无序列表

无序表表示**没有刻意顺序**的列表，无序列表使用父标签`<ul></ul>`标签（unordered list），每个列表项都是子标签`<li></li>`标签(list item)（`<li>`子标签不能单独使用吗，它必须放到,`<ui>或`者`<ol>`中使用），无序标签是一个父子组合标签，**上阵父子兵，不能单独出现**，`<ul>`标签的子标签只能是<li>标签。比如：

```html
<ul>
    <li>面包</li>
    <li>牛奶</li>
    <li>鸡蛋</li>
    <li>水果</li>  
</ul>
```

**`<ul>`标签的子标签只能是`<li>`标签。**比如：![image-20210606140111511](https://xiaodingyang-1300707163.cos.ap-chengdu.myqcloud.com/Markdown/image-20210606140111511.png)

`<li>`标签是容器，内部可以放任何标签如：

```html
<ui>
    <li>面包</li>
    <li>牛奶
        <p>要脱脂或低脂的 </p>
    </li>
    <li>鸡蛋</li>
    <li>水果 </li>
</ui>
```

注意：当html标签形成嵌套，必须注意代码的缩进（Tab键）

### 有序列表

有序列表是**有刻意的**列表，使用`<ol></ol>`标签（ordered list)，每个列表项都是`<li><li>`标签

> `<ol>`标签注意事项
>
> 1. li 标签不能散着放，它必须是 ol 标签或者 ul 标签的子标签
> 2.  ol 的子标签只能是 li 标签
> 3.  li 中可以放任何东西

**`<ol>`标签的type属性**

| type属性值 | 意义                   |
| :--------: | :--------------------- |
|     a      | 表示小写英文字没有编号 |
|     A      | 表示大写英文字母编号   |
|     i      | 表示小写罗马数字编号   |
|     I      | 表示大写罗马数字编号   |
|     1      | 表示数字编号（默认）   |

**`<ol>`标签staet属性**

start属性值必须是一整数，指定列表编号的起始值

此属性的值应为**阿拉伯数字**，尽管列表条目的编号类型type属性可能指定为罗马数字编号等其他类型的编号

```html
<body>      
    </h1> 考试成绩排名</h1>
    <ol tipe="a" staet="3" reversed >
        <li>小穆</li>
        <li>小红</li>
        <li>小红</li>
    </ol>
</body>
```





**`<ol>`标签的reversed属性**

reversed属性指定列表中的条目是否为倒序排列

reversed属性不需要值，只需要书写reversed单词即可，这是HTML的全新属性

### 多媒体与语义标签

#### 图片与超链接标签

##### 1图片标签

<img>标签用来在网页钟插入图片

##### 

```html
<img> src="images/gugong.jpg">
<!--img是images图片的缩写，src是source来源的缩写，images/表示inages文件夹，gugong是图片的存储目录和完整文件名，后缀名jpg要写-->
```

一定要注意，图片必须复制到项目的文件夹中，一般将图片保存到项目文件夹中的images子文件夹中、

图片本质上没有被插入到网页中，只是被引入到了网页中。所以将来要讲图片也一起上传到服务器上，将图片复制到项目文件中，即可整体上传。

**`<img>`标签的alt属性**

1.alt属性是alternate“替代品”的缩写，他是对图片文本的描述，不是强制性的

```html
<img src="inages/gugong.jpg" alt="故宫角楼">
```

2.如果由于某种原因无法加载图片，浏览器会在页面显示alt属性中的备注文本

3.供视力不方便的朋友使用网页朗读器，也会朗读alt中的文本

**`<img>`标签的width、height属性**

width、height属性分别设置宽度和高度，单位是像素，但是不需要写单位。一般只写一个属性，对应的宽高默认比例。

```html
<img src="images/gugong.png" alt="故宫角楼"width="300">
```

网页支持的图片格式

|     格式      | 说明                                                     |
| :-----------: | -------------------------------------------------------- |
|     .bmp      | windows画图软件默认保存的格式，位图                      |
|     .gif      | 支持动画（比如表情包）                                   |
| .jpeg（.jpg） | 有损压缩图片，用于照片                                   |
|     .png      | 便携我是网络图像，用于logo，背景图形等，支持透明和半透明 |
|     .svg      | 矢量图片                                                 |
|     .webp     | 最新的压缩算法非常优秀的图片格式                         |

**相对路径**

相对路径描述从网页出发，如何找到图片

随着网页和图片的位置关系不同，插入图片的代码随之改变

如果需要回退层次，使用"../"这样的写法，回退2层写2个"../"

```HTML
<img src="../inages/gugong.jpg" alt="故宫角楼">
```

**绝对路径**

绝对路径描述图片精准地址

```HTML
<img src="图片的网址"
```

不管网页在哪里，绝对路径
